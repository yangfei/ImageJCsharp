# ImageJ CSharp MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first usable ImageJ-like MVP with a C# core library and a WinForms shell.

**Architecture:** Keep image data, ROI, measurements, and processing algorithms in `ImageJCsharp.Core` so they can be tested without UI. Keep WinForms responsible for menus, image display, dialogs, and command wiring.

**Tech Stack:** .NET Framework 4.8 for app, core library, and tests; WinForms, xUnit, pure C# image buffers for MVP processing.
