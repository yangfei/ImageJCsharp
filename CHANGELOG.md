# Changelog

All notable changes to ImageJCsharp will be documented in this file.

The project uses semantic versioning before `1.0.0` in a practical way:

- `0.x.0` versions may change APIs and scope.
- Patch versions fix bugs or documentation.
- `1.0.0` means the first stable ImageJ-like core replacement target has been reached.

## Unreleased

### Added

- Initial project documentation.
- Roadmap and near-term plan.
- .NET Framework 4.8 solution structure.
- Core grayscale image model.
- Rectangle ROI model.
- Basic measurement model.
- Threshold, invert, and Sobel edge processing.
- Initial WinForms shell.
