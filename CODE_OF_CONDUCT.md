# Code of Conduct

ImageJCsharp is intended to be a welcoming project for developers, researchers, students, and users interested in scientific image analysis.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Explain technical disagreements clearly.
- Help newcomers understand the project.
- Keep discussions focused on the project and its users.

## Unacceptable Behavior

- Harassment or personal attacks.
- Insults, threats, or discriminatory language.
- Repeated off-topic disruption.
- Publishing private information without permission.

## Reporting

If a discussion becomes harmful, open a private communication channel with the project maintainer if one is available. Until the project has a formal maintainer contact, use GitHub moderation tools and keep public discussion focused on technical resolution.

## Enforcement

Maintainers may remove comments, close issues, or block participants who harm the project community.
